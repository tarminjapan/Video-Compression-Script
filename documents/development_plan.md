# AmeCompression GUI Development Plan

## 概要

既存のCLI動画・音声圧縮ツールに **CustomTkinter** ベースのGUIを追加し、**ダークモード/ライトモード切替**、**日本語/英語UIの両方対応**、**PyInstallerによる.exe配布**を実現する。

## Milestone 一覧

| Milestone | タイトル | 目標 |
| ----------- | --------- | ------ |
| **M1** | プロジェクト基盤構築 | CustomTkinter導入、i18n基盤、基本ウィンドウ表示 |
| **M2** | コアGUI機能 | ファイル選択、圧縮設定パネル、進捗表示 |
| **M3** | 応用機能 | バッチ処理UI、音量調整、ノイズ除去 |
| **M4** | 設定・外観 | ダーク/ライトモード、設定画面、i18n全画面対応 |
| **M5** | 配布・品質保証 | PyInstaller .exe化、テスト、ドキュメント整備 |

## Phase 1: プロジェクト基盤構築（Milestone M1）

**概要:**

**タスク:**

- [ ] `customtkinter` を `requirements.txt` / `pyproject.toml` に追加

## 依存関係

### 実行時

- `customtkinter` >= 5.2
- `Pillow` >= 9.0（CustomTkinter依存）

### 開発時

- `pyinstaller` >= 6.0
- `pytest` >= 7.0
- `ruff`（フォーマッター & リンター）

## 優先順位と着手順

1. **Phase 1** → まず基本ウィンドウが立ち上がる状態を目指す
2. **Phase 2** → 1ファイルの圧縮がGUIからできる状態（MVP）
3. **Phase 3** → 応用機能で使いやすさを向上
4. **Phase 4** → 見た目と設定の完成度を高める
5. **Phase 5** → 配布可能な .exe を完成させる

**最初の目標:** Phase 2完了時点で「GUIから1つの動画ファイルを圧縮できる」状態を作る。
